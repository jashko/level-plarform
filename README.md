# LEVEL Platform — день 1

Аналитическая платформа для девелопера. Скоринг 14 городов-миллионников РФ + финансовое моделирование проектов.

## Что появилось в этой версии

✅ **Главный экран с рейтингом 14 городов** — открываешь приложение и сразу видишь куда идти, никакого ручного ввода
✅ **Карточка города** — все подскоры, демография, рынок жилья, конкуренция, источники данных
✅ **Связка «город → финмодель»** — клик и в финмодель уже подставлены цена м² и темп продаж этого города
✅ **Автоматическое подтягивание ставки ЦБ** — единственный реально автоматический источник, остальное — снимок

## 14 городов и где взяты данные

Новосибирск, Екатеринбург, Казань, Нижний Новгород, Челябинск, Самара, Уфа, Ростов-на-Дону, Омск, Краснодар, Воронеж, Волгоград, Пермь, Красноярск.

**Источники:**
- Население — Росстат, на 1 января 2025
- Зарплаты — РИА Рейтинг по данным Росстата за январь–июнь 2025
- Цены м² — bnMAP.pro январь 2026, РБК, Яндекс Недвижимость
- Объём сделок, рост цен — Циан / РБК / Коммерсант, итоги 2025
- Ключевая ставка / ипотека — **ЦБ РФ напрямую**, fallback на снимок 27.04.2026
- КРТ-программы, «белые пятна» — экспертная оценка, **требует верификации аналитиком Level**

## Установка и запуск

```bash
npm install
npm run ui
```
Откроется http://localhost:3000

При загрузке приложение тянет данные ЦБ и считает рейтинг. На первом экране — таблица 14 городов с CityScore, зонами, ценами м². Клик по строке → карточка города. Внизу карточки кнопка «Финмодель» → переход в расчёт IRR/NPV с уже подставленными параметрами этого города.

## Структура

```
src/
├── data/
│   ├── cities.ts          # датасет 14 городов с источниками
│   └── ranking.ts         # оркестратор: ЦБ → MacroScore → CityScore × 14
├── engine/
│   ├── datasources/
│   │   └── cbr.ts         # реальный fetcher Банка России
│   ├── finance/           # финансовый движок (эскроу + ПФ + NPV/IRR)
│   └── scoring/           # 4 уровня скоринга (Macro/City/District/Site)
ui/
├── index.html
├── app.js                 # 3 экрана: главная / карточка города / финмодель
└── engine.js              # бандл (48.9 KB), генерируется npm run bundle
```

## Команды

## Честное про автоматизацию

Полностью автоматическое подтягивание данных требует месяцев работы и партнёрских API. В текущей версии:
- **Автоматически** обновляется только ставка ЦБ (открытый XML-фид)
- **Снимок на дату 2026-02-01** — всё остальное: цены, зарплаты, объёмы сделок. В проде заменяется скрапером Росстата (раз в квартал) и API Дом.РФ (раз в неделю)
- **Ручной ввод аналитиком** — КРТ-программы, конкуренты, белые пятна. Этого в открытых источниках нет.

При защите курсового это нормальная архитектура — все три уровня автоматизации честно показаны и обоснованы.
